# waterfall
jquery的瀑布流插件，每个子元素宽度相同高度不同，在页面当中实现参差的排列，即瀑布的形式排列

采用ajax异步请求，每点击一次"正在加载"，都会获取到存到json里的数据，把数据通过模板引擎封装好，在页面上呈现出瀑布流的排列效果
